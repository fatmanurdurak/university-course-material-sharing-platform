# University Course Material Sharing Platform

A web-based platform that allows university students and instructors to share course materials, assignments, and academic resources.

## Features

* User registration and login system
* Instructor dashboard to create and manage courses
* Upload and share course materials
* Assignment submission system for students
* Grade management for instructors
* Notification system for updates and submissions

## Technologies Used

* PHP
* HTML
* CSS
* MySQL (via phpMyAdmin / XAMPP)

## Project Structure

* `index.php` – Main homepage
* `login.php / register.php` – User authentication
* `dashboard.php` – Instructor dashboard
* `upload_material.php` – Upload course materials
* `submit_assignment.php` – Assignment submission system
* `grade_assignment.php` – Instructor grading system
* `config.php` – Database configuration

## Purpose of the Project

This project was developed as part of a university Web Design course to create a functional platform for sharing course materials and managing assignments between students and instructors.
